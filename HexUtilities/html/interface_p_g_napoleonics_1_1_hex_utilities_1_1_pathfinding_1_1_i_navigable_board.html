int PGNapoleonics.HexUtilities.Pathfinding.INavigableBoard.Heuristic ( int  range)

Returns an A* heuristic value from the supplied hexagonal Manhattan distance range.

Returning the supplied range multiplied by the cheapest movement cost for a single hex is usually suffficient. Note that heuristic must be monotonic in order for the algorithm to perform properly.
