Detailed Description

A BoardStorage implementation optimized for large maps by blocking 32 x 32 arrays of hexes for improved caching.

This BoardStorage implementation stores the board cells in blocks that are 32 x 32 cells to provide better localization for the Path-Finding and Field-of-View algorithms.
