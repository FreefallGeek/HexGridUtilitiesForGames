Detailed Description

Provides a thread-safe priority queue data structure.

Template Parameters
TKeySpecifies the type of keys used to prioritize values.
TValueSpecifies the type of elements in the queue.

The implementation is slightly modified from that posted on the Microsoft site, by the change to a custom HexKeyValuePair and the implementation of the interface IPriorityQueue to facilitate interchane with DictionaryPriorityQueue at shorter ranges.

Returns an enumerator that iterates through the collection.

Returns
An enumerator for the contents of the queue.

The enumeration represents a moment-in-time snapshot of the contents of the queue. It does not reflect any updates to the collection after GetEnumerator was called. The enumerator is safe to use concurrently with reads from and writes to the queue.
