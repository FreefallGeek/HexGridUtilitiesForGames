Detailed Description

Stable (insertion-order preserving for equal-priority elements) PriorityQueue implementation.

Eric Lippert's C# implementation of PriorityQueue for use by the A* algorithm.

<cref>http://blogs.msdn.com/b/ericlippert/archive/2007/10/08/path-finding-using-a-in-c-3-0-part-three.aspx</cref>
